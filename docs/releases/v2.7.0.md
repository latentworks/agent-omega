failure, and is recorded here so the release does not present that job as green.

## Scope not yet claimed

- Protocol mechanics, lifecycle durability, and package provenance are verified.
  They are not evidence by themselves that the harness improves model quality.
- The 12-hour factual capability evaluation is specified in
  [`docs/plans/2026-07-11-v2.7-capability-evaluation.md`](../plans/2026-07-11-v2.7-capability-evaluation.md)
  and will report supported, unsupported, or indeterminate claims from saved
  run artifacts.
- No Linux desktop shell is introduced. The Linux assets are engine-only.
- Native assets are unsigned; macOS signing and notarization were not performed.
