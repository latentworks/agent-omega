This is a skill invocation. Invoke the 'skill-creator' skill with your skill tool, then follow it to author a new skill for the topic below.

Save the result as $XDG_CONFIG_HOME/opencode/skill/<name>/SKILL.md (or ~/.config/opencode/skill/<name>/SKILL.md if XDG_CONFIG_HOME is unset) so OpenCode auto-discovers it, and optionally add a matching command at $XDG_CONFIG_HOME/opencode/command/<name>.md (same ~/.config fallback) so the user can force it with /<name>. The new skill appears only after the engine restarts.
